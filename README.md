# HelloPlugin

A simple Paper plugin that greets players with commands and a welcome title on join.

## Commands

| Command | Description |
|---|---|
| `/hello` | Greets you privately |
| `/hello world` | Broadcasts your greetings to everyone |
| `/hello <player>` | Sends greetings to a specific player |

## Download

Go to the [Releases](../../releases/latest) page and download the latest `.jar`.

## Requirements

- Minecraft 26.1.2
- Paper build #60+
- Java 25

## Installation

1. Download the `.jar`
2. Place it in your server's `plugins/` folder
3. Restart the server
