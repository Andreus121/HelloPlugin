# HelloPlugin

Plugin sencillo para Paper que saluda a los jugadores mediante comandos y un título de bienvenida al conectarse.

## Comandos

| Comando | Descripción |
|---|---|
| `/hello` | Te saluda a ti en privado |
| `/hello world` | Manda tus saludos a todos en el servidor |
| `/hello <jugador>` | Saluda a un jugador específico |

## Requisitos

- Minecraft 26.1.x
- Paper build 60+
- Java 25

## Instalación

1. Descarga el `.jar`
2. Colócalo en la carpeta `plugins/` de tu servidor
3. Reinicia el servidor
